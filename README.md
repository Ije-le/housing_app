# Housing App

This repository now includes a small Flask app for browsing grouped items and their meeting mentions.

## Run locally

```bash
python app.py
```

Then open http://127.0.0.1:5000/ in your browser.

## What it uses

- `extracted_json/all_items_grouped.csv` for the category groups and counts
- `extracted_json/all_items_summary.json` for per-item mention details and dates
